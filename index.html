üöÄ Usage

  1. Open an email
  2. Go to the Apps tab and search for AI Email Helper
  3. Right-click it and select Pin for easier access
  4. Click the AI Email Helper button in the Ribbon under Apps
  5. The task pane opens to the right
  6. Configure your API key, Provider, Model, Base URL (if needed)
  7. Click Summarise or Draft Reply or Fine-tune Reply to use the Helper

üìù Notes

  • You need to be in Compose mode to insert the drafted reply into the message body.
  • When in Compose mode, click into the body of the last email message so the Helper can access its text.
